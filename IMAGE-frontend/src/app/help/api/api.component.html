API Help Page

1. There are two main entrypoints:
one for organism: https://wp5image.eu/data_portal/backend/organism/
one for specimen: https://wp5image.eu/data_portal/backend/specimen/

2. To get list of all organisms:
API format
curl https://wp5image.eu/data_portal/backend/organism/?format=api
JSON format
curl https://wp5image.eu/data_portal/backend/organism/?format=json

3. To get information about specific organism:
API format
curl https://wp5image.eu/data_portal/backend/organism/SAMEA5159526/?format=api
JSON format
curl https://wp5image.eu/data_portal/backend/organism/SAMEA5159526/?format=json

4. To get list of all specimens:
API format
curl https://wp5image.eu/data_portal/backend/specimen/?format=api
JSON format
curl https://wp5image.eu/data_portal/backend/specimen/?format=json

5. To get information about specific specimens:
API format
curl https://wp5image.eu/data_portal/backend/specimen/SAMEA5159523/?format=api
JSON format
curl https://wp5image.eu/data_portal/backend/specimen/SAMEA5159523/?format=json
