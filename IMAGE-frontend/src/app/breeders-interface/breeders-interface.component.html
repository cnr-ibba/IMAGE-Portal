Breeders Interface - MoBPS

Modular Breeding Program Simulator

MoBPS is an R-package (https://github.com/tpook92/MoBPS) and graphical interface (available at www.mobps.de) to simulate complex and large scale breeding programs with focus on livestock and crop populations. Simulations are performed on an individual basis. MoBPS is a versatile tool, providing standard procedures applied in animal and plant breeding like GBLUP and OGC, but also allowing to use user provided own selection schemes while still controlling the simulation of phenotypes, meiosis and costs of the simulated scheme.

Information about how to use IMAGE data with this package could be found here (under Guidelines to the R-package section)
